4) What is the purpose of align property in table cells?

The `align` attribute specifies the horizontal alignment of content within a cell (`left`, `center`, or `right`), while `valign` specifies vertical alignment. Modern HTML uses CSS (`text-align` and `vertical-align`).


5) Is cellpadding a valid property?

`cellpadding` is outdated in HTML5, but still works to control spacing within cells. CSS (`padding` within `td` or `th`) is now preferred.


6) How to expand a cell to two rows and three columns?

Use `rowspan="2"` and `colspan="3"` together.
